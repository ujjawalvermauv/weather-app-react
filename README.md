# 🌦️ Weather App  

A sleek and user-friendly weather app that provides real-time weather updates for any city worldwide. Stay informed with accurate data at your fingertips!  

---

## 🌟 Features  

- 🌍 **Global Access**: Get weather updates for any city.  
- 📊 **Detailed Insights**: View temperature, humidity, wind speed, and more.  
- 🎨 **Modern UI**: Clean and responsive design for all devices.  
- 🔄 **Real-Time Updates**: Fetches the latest weather data instantly.  





🔗 Built With
Frontend: HTML, CSS, JavaScript
API: OpenWeatherMap API
Tools: Axios for HTTP requests

🔗 API Integration
This app uses the OpenWeatherMap API to fetch real-time weather data.
